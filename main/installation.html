Installation

Installing req2flatpak is easy. Multiple installation methods are available.

Installing the Python Package

You can install req2flatpak as a python package. This is the default and recommended installation method.

For example:

pip install git+https://github.com/johannesjh/req2flatpak
req2flatpak --help

The first command will install the latest development version of req2flatpak. The second command will display req2flatpak’s help, allowing you to verify that req2flatpak was installed successfully.

Simply Downloading or Copying the Script

You can download the req2flatpak script to your computer and run it. Simple as that.
